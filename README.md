Calculadora de Médias Acadêmicas

Contexto
A Calculadora de Médias Acadêmicas é uma aplicação web desenvolvida para automatizar o cálculo de notas e médias escolares, reduzindo erros manuais e tornando o acompanhamento do desempenho acadêmico mais simples e intuitivo. O projeto foi criado como forma de aplicar conceitos fundamentais de desenvolvimento web e lógica de programação em uma solução prática para estudantes.
Desenvolvimento
O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, com foco em simplicidade, clareza visual e facilidade de uso. A aplicação permite que o usuário informe notas e obtenha automaticamente os resultados dos cálculos, apresentando as informações de forma organizada e acessível.
Entre as principais funcionalidades implementadas estão:
Inserção dinâmica de notas;
Cálculo automático de médias;
Validação de dados informados pelo usuário;
Atualização dos resultados em tempo real;
Exibição de status de aprovação ou reprovação com base nos critérios definidos;
Interface responsiva para diferentes dispositivos.
Durante o desenvolvimento, foram aplicados conceitos de manipulação do DOM, eventos JavaScript, estruturas condicionais, operações matemáticas e organização de código voltada à manutenção e legibilidade.
Resultados
O resultado foi uma aplicação funcional capaz de automatizar cálculos acadêmicos de forma rápida e confiável. O projeto contribuiu para o fortalecimento de conhecimentos em lógica de programação, desenvolvimento front-end e criação de interfaces interativas, além de demonstrar a aplicação prática de conceitos fundamentais da engenharia de software.
